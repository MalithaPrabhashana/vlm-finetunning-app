# vlm-finetunning-app

## Setup

### 1. Create Virtual Environment

```bash
python -m venv venv
```

### 2. Activate Virtual Environment

#### On Windows

```bash
venv/Scripts/activate
```

#### On macOS / Linux / WSL

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```
